# Todo_programmers

<br>해당 프로젝트를 clone 한후
app.js가 있는 폴더에서 npm install 명령어를 입력한 후 필요한 모듈을 설치합니다.
<br>
__node app.js__ 또는
__npm start__ 로 실행이 가능하며,
로컬 접속은
__localhost:3000__ 으로 가능합니다!
